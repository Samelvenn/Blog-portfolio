Résumé du projet

Ce projet consiste à réaliser une automatisation et une sécurisation de la gestion des utilisateurs à l'aide de différents programmes.

Objectifs

-Mettre en place une base de données pour stocker les utilisateurs et leurs rôles.

-Développer des programmes pour permettre la gestion des utilisateurs.

-Mettre en place un protocole pour la gestion des différents outils développés.

Ce que j'ai réalisé

-Extraction des données des utilisateurs via un script Powershell sur des fichiers CSV.

-Création de script Python pour extraire les données des utilisateurs depuis des fichiers CSV.

-Génération de mots de passe aléatoires pour les nouveaux utilisateurs via des scripts Python.

-Implémentation dans une base de données SQL.

-Réalisation de tests et corrections des différents bugs survenus.

Les outils utilisés

Pour se projet nous avons utilisé Python, SQL, Powershell et les fichiers csv. Nous avons manipulé les outils suivants qui sont MariaDB, VScode, ActiveDirectory ainsi que des machines virtuelles.

Résultats

Le projet fournit une interface opérationnelle pour gérer les comptes utilisateurs et illustre les bonnes pratiques de conception et de sécurité mises en œuvre.

Compétences acquises

-Compétences en gestion de bases de données SQL.
